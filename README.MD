## 如何编译本项目

### 环境要求

```

~$ lsb_release -a
Distributor ID:	Ubuntu
Description:	Ubuntu 18.04.3 LTS
Release:	18.04
Codename:	bionic

```

### 依赖库
本项目依赖于readline
> 安装方法

```
    sudo apt-get install libreadline-dev
```
在Makefile 所在目录下执行
```
make
```
## 运行
在编译完成后可在bin目录下找可执行程序
```
./bin/gethost
```
